# 🏦 Family Budgeting App (Django MVP)

A simple one-page budgeting web app built with Django + SQLite.

## Features
- 50/30/20 Budget Calculator  
- Manage Funds (Money In / Out)  
- Track Debts & Bills  
- Add Family Members  
- Simulated Chatbot
